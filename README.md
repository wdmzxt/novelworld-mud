# NovelWorld MUD

Turn novels into playable AI-powered text adventure worlds.

NovelWorld MUD is a local single-player text adventure project. The v0.1 goal is to establish a clean command-line MUD skeleton that can later grow into an AI-powered interactive fiction engine.

## v0.1 Scope

- Local command-line play loop
- Basic command planning
- Demo world data in JSON
- No AI integration
- No ChromaDB
- No SQLite

## Planned Commands

- `look`
- `go <地点>`
- `talk <人物>`
- `take <物品>`
- `bag`
- `status`
- `help`
- `quit`

## Run

```bash
python main.py
```
