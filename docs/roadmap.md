# NovelWorld MUD Roadmap

Current official version: v0.8.

Do not label the current project as v0.8.1. The next planned milestone is v0.9.

## Completed

- v0.1 Basic playable MUD prototype
- v0.2 JSON save/load system
- v0.3 Basic quest system
- v0.4 Basic event system
- v0.5 Data-driven quests and events
- v0.6 World schema validation
- v0.7 Expanded demo world content
- v0.8 Bilingual language support

## Future Plan

- v0.9 Optional LLM narration
- v1.0 Novel import prototype
- v1.1 GUI prototype
- v1.2 Novel-to-world extraction

## Notes

The project remains local-first. AI narration, vector memory, and GUI features are future optional layers, not requirements for the current v0.8 engine.
